## 详细步骤说明

### 1️⃣ 数据准备

| 输入文件 | 说明 |
|---------|------|
| Query 基因组 | 待分析的基因组序列 (FASTA) |
| Reference 基因组 | 同物种参考基因组 (FASTA) |
| 参考基因组注释文件 | GFF/GTF (用于基因注释) |

---

### 2️⃣ 序列比对

**方法 A：有测序 reads 的情况**
```bash
# BWA 比对
bwa index ref.fasta
bwa mem ref.fasta reads_1.fq reads_2.fq > alignment.sam

# 排序与去重
samtools sort alignment.sam > alignment_sorted.bam
samtools rmdup alignment_sorted.bam alignment_rmdup.bam
```

**方法 B：两序列直接比对（无测序数据）**
```bash
# 使用 MUMmer 或 BLAST
nucmer ref.fasta query.fasta
show-snps -Clr ref.query.delta > snps.txt
```

---

### 3️⃣ 变异检测

```bash
# 方法1: bcftools (轻量)
bcftools mpileup -f ref.fasta alignment_rmdup.bam | \
bcftools call -mv -Ob -o variants.bcf

# 方法2: GATK (标准流程)
gatk HaplotypeCaller -R ref.fasta -I alignment_rmdup.bam \
    -O variants.vcf
```

---

### 4️⃣ 变异过滤

```bash
# bcftools 过滤
bcftools filter -Oz -o variants_filtered.vcf.gz \
    -s LOWQUAL -e 'QUAL < 30 || QD < 2.0 || MQ < 40.0'

bcftools index variants_filtered.vcf.gz
```

---

### 5️⃣ 功能注释

```bash
# SnpEff 注释
snpEff ann -v GRCh38.105 variants_filtered.vcf > annotated.vcf

# snpSift 提取关键信息
snpSift extractFields annotated.vcf CHROM POS REF ALT ANN[0].EFFECT \
    ANN[0].GENE ANN[0].IMPACT > mutations_summary.tsv
```

---

### 6️⃣ 生物学意义分析

| 分析类型 | 工具/方法 | 解释意义 |
|---------|----------|---------|
| **突变类型** | SnpEff | 同义/非同义、移码、终止密码子突变 |
| **选择压力** | dN/dS (PAML) | 基因是否受正向/负向选择 |
| **有害性预测** | SIFT, PolyPhen-2 | 突变对蛋白功能的影响 |
| **保守性** | PhyloP, GERP++ | 位点在进化中的保守程度 |
| **通路富集** | KEGG, Reactome | 突变是否影响特定生物学通路 |

## 📌 注意事项

1. **细菌/病毒基因组**：可用 ` Prokka` 注释 + `BLAST` 比对
2. **无参考基因组**：需要先 **de novo 组装**，再进行变异检测
3. **大基因组（人类/动植物）**：建议使用 GATK Best Practices 标准流程
